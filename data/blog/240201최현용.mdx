`callback`

```jsx
function a(){
	console.log('A');
}

var a = function(){ // 익명함수
	console.log('A');
}

a();
```

`JavaScript`에서는 함수가 값이다.

```jsx
var a = function(){ // 익명함수
	console.log('A');
}

function slowfunc(callback){
	callback();
}

slowfunc(a);
```
